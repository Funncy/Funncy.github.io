### 나의 풀이

---

```jsx
function solution(s) {
  let count = 0;
  const list = s.split("");
  for(let i = 0; i <list.length ; ++i) {
    if(list[i] == "(") {
      count++;
    }else if(count <= 0) {
      return false;
    }else {
      count--;
    }
  }

  return count == 0;
}
```

시간내에 빨리 들어오기 위해 for 문을 활용해 풀어보았다.

위와 같은 로직이지만 다른 분들 풀이중에 더 깔끔한 코드가 있었다.

```jsx
function solution(s){
    let cum = 0
    for (let paren of s) {
        cum += paren === '('? 1: -1
        if(cum < 0) {
            return false
        }
    }
    return cum === 0? true: false;
}
```

### 포인트

1. for of 를 활용해 split을 하지 않았다.
2. 삼항 연산자를 통해 count 연산을 한줄로 만들었다.
